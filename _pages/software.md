## Learning to Optimize

| Algorithm | Description |
|-----------|-------------|
| [**IPM-LSTM**](https://github.com/NetSysOpt/IPM-LSTM) | A learning-based interior point method for nonlinear programming |
| [**SymILO**](https://github.com/NetSysOpt/SymILO) | A symmetry-aware learning framework for integer linear optimization |
| [**ROS**](https://github.com/NetSysOpt/ROS) | A GNN-based relax-optimize-and-sample framework for Max-k-Cut |
| [**Predict-and-Search**](https://github.com/sribdcn/Predict-and-Search_MILP_method) | A GNN-guided predict-and-search framework for mixed-integer linear programming |
